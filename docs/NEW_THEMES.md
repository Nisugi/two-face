## WCAG Compliance

### Understanding Contrast Ratios

**WCAG 2.1 Standards**:
- **Level AA**: Minimum 4.5:1 for normal text, 3:1 for large text
- **Level AAA**: Minimum 7:1 for normal text, 4.5:1 for large text

### Compliance by Theme Category

#### General Purpose Themes
- **Compliance Level**: AA minimum
- Most themes achieve 7:1+ contrast ratios
- Links may use 4.5:1 (acceptable for AA)

#### High Contrast Themes
- **Compliance Level**: AAA
- 21:1 contrast ratio (maximum possible)
- Suitable for severe low vision

#### Colorblind Themes
- **Compliance Level**: AA minimum
- Use luminance contrast as primary differentiator
- Avoid problematic color combinations
- Include non-color cues (underlines, borders)

#### Photosensitivity Themes
- **Compliance Level**: Custom criteria
- Maximum brightness limits
- Low saturation requirements
- Designed to minimize triggers

### Testing Recommendations

1. **Contrast Testing**: Use tools like [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/)
2. **Colorblind Simulation**: Test with tools like [Coblis](https://www.color-blindness.com/coblis-color-blindness-simulator/)
3. **User Testing**: Get feedback from users with actual accessibility needs
4. **Automated Testing**: Integrate accessibility linters in CI/CD

### Registering the Theme

In the `ThemePresets::all()` method:

```rust
pub fn all() -> HashMap<String, AppTheme> {
    let mut themes = HashMap::new();

    // Existing themes
    themes.insert("dark".to_string(), Self::dark());
    themes.insert("light".to_string(), Self::light());
    // ... other existing themes ...

    // Add new themes
    themes.insert("ocean_depths".to_string(), Self::ocean_depths());
    themes.insert("forest_canopy".to_string(), Self::forest_canopy());
    themes.insert("sunset_boulevard".to_string(), Self::sunset_boulevard());
    // ... etc ...
```

### Converting Hex to RGB

Utility function (already exists in theme.rs):

```rust
// Convert hex string to Color
fn hex_to_color(hex: &str) -> Color {
    let hex = hex.trim_start_matches('#');
    let r = u8::from_str_radix(&hex[0..2], 16).unwrap_or(0);
    let g = u8::from_str_radix(&hex[2..4], 16).unwrap_or(0);
    let b = u8::from_str_radix(&hex[4..6], 16).unwrap_or(0);
    Color::Rgb(r, g, b)
}

// Example usage:
window_border: hex_to_color("#1e4d6b"),
```

- [ ] Colors work for target accessibility group (if applicable)
